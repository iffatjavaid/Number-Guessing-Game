🎯 Number Guessing Game
📌 Description
This is a console-based Number Guessing Game developed in Python.
The program randomly generates a number, and the player has to guess it. The game provides hints whether the guess is too high or too low until the correct number is guessed. It also displays the total number of attempts and allows the player to play multiple rounds.
✨ Features
Random number generation
Unlimited guessing attempts
Too High / Too Low hints
Attempt counter
Play Again option
User-friendly console interface
🛠 Technologies Used
Python 3
📚 Concepts Used
Variables
Input/Output
If-Else Statements
While Loop
Random Module
Comparison Operators
Break Statement
Integer Type Casting
▶️ How to Run
Run the following command:
python NumberGuessingGame.py
